================================================================================
ICON USAGE RULES FOR AI CHAT COMMUNICATION
================================================================================

This document defines comprehensive rules for using icons/emojis in AI chat
responses. These rules ensure consistent, clear, and effective visual
communication while maintaining professionalism.

All comments and examples in this file are in English.
User communication should be in German (as per communication rules).

================================================================================
SECTION 0: RULE PRIORITY SYSTEM & CONFLICT RESOLUTION
================================================================================

PRIORITY LEVELS:
- P0 (Critical): Mandatory icon usage for specific contexts
 These rules must never be violated. Icons are required for clarity and
 user experience.

- P1 (High): Recommended icon usage for enhanced communication
 These rules improve communication quality and should be followed consistently.

- P2 (Medium): Optional but beneficial icon usage
 These rules enhance readability but are not strictly required.

- P3 (Low): Decorative icon usage
 These rules are nice-to-have enhancements.

CONFLICT RESOLUTION:
When icon usage conflicts with other rules:
1. P0 > P1 > P2 > P3 (higher priority always wins)
2. Clarity > Decoration (functional icons over decorative)
3. Consistency > Variety (consistent icon usage patterns)
4. Context-appropriate > Generic (icons should match context)
