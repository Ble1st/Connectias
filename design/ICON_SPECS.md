# App Icon Design Specifications

## Overview
The new app icon for **Connectias** represents the core pillars of the application: Security, Connectivity, Power, Hardware, and Privacy. The design utilizes a modern "star diagram" (radar chart) metaphor with the "C" brand mark at the center, symbolizing the app as the central hub for these metrics.

## Visual Style
- **Style:** Flat, Modern, Geometric.
- **Grid:** 48x48px base grid (scalable vector).
- **Format:** SVG (Scalable Vector Graphics).

## Color Palette
The design strictly adheres to the defined palette:

| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Connectias Blue** | `#4A90E2` | Primary Brand Color. Used for the central hub, diagram lines, and strokes. |
| **Pure White** | `#FFFFFF` | Backgrounds, the "C" logo, and negative space in icons. |
| **Alert Gold** | `#F5A623` | Accent Color. Used for the feature symbols at the tips of the star to indicate activity/monitoring. |

## Structure
1.  **Background:** A subtle circular container (White/Off-White gradient) with a thin Blue stroke.
2.  **Central Hub:** A solid Blue (`#4A90E2`) circle containing the White (`#FFFFFF`) "C" logomark.
3.  **Diagram:** Five spokes radiating from the center, forming a pentagonal connection web.
4.  **Feature Symbols (Tips):**
    *   **Top (0°):** Shield (Security/Vulnerability Scanner)
    *   **Top Right (72°):** Signal/Wifi (Network Analysis)
    *   **Bottom Right (144°):** Lightning Bolt (Battery Analyzer)
    *   **Bottom Left (216°):** Microchip (Hardware/USB)
    *   **Top Left (288°):** Eye (Privacy Dashboard)

## Usage
This SVG is designed to be exported as:
- Android Adaptive Icon (Foreground layer).
- Web Favicon.
- Marketing material assets.
